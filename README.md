# Whack A Mole
### Fannar Þór Valdimarsson og Freyr Þórhallson
spilið er whack a mole og það er hægt að spila 2 saman. spilið virkar þannig að það kveikir á tökkum og þú átt að reyna að ýta á hann á undan hinni manneskjunni. sem ýtti á flesta takka vinnur umferðina og fær refsingu svo hinn fær meiri séns á að vinna. Fyrstur til að vinna 3 umferðir vinnur allt.
